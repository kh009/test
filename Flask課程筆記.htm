 1.何謂Flask?:

 

Flask 是一個用 Python 編寫的 Web 應用程序框架。Armin Ronacher 開發了它，他領導了一個名為 Pocco 的國際 Python 愛好者小組。Flask 基於 Werkzeug WSGI 工具包和 Jinja2 模板引擎。

兩者都是 Pocco 項目。

3.Flask – Application(應用)

範例:存成hello.py

from flask import Flask

app = Flask(__name__)

 

@app.route('/')

def hello_world():

   return 'Hello World’

 

if __name__ == '__main__':

   app.run()

4.Flask – Variable Rules(變量規則)

from flask import Flask

app = Flask(__name__)

 

@app.route('/hello/<name>')

def hello_name(name):

   return 'Hello %s!' % name

 

if __name__ == '__main__':

   app.run(debug = True)

這是字串

有這些宣告類型:

範例程式:

from flask import Flask

app = Flask(__name__)

 

@app.route('/blog/<int:postID>')     

def show_blog(postID):

   return 'Blog Number %d' % postID  //整數

 

@app.route('/rev/<float:revNo>')

def revision(revNo):

   return 'Revision Number %f' % revNo  //浮點數

 

if __name__ == '__main__':

   app.run()

11.Flask Session—會畫

和 Cookie不一樣，Session 數據也存儲在伺服器端。會話是客戶端登錄服務器和註銷服務器的時間間隔。需要在整個會話中保留的數據存儲在客戶端瀏覽器中。與每個客戶端的會話都分配有一個Session ID。會話數據存儲在 cookie 之上，服務器以加密方式對其進行簽名。對於這種加密，Flask應用程序需要一個已定義SECRET_KEY。
